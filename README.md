## Estrutura de arquivos

```
portfólio
├─ doc
|  ├─ wireframe.pdf
├─ src
|  ├─ static
|  |  ├─ css
|  |  |  └─ arquivos (.css)
|  |  ├─ img
|  |  |  └─ imagens (.png)
|  |  ├─ js
|  |  |  └─ main.js  
|  ├─ templates
|  |  └─ arquivos (.html)
|  └─ app.py
├─ .gitignore
├─ README.md
└─ requirements.txt
 ```
### Descrição das pastas
 > **Pasta doc:** possui o wireframe do projeto

 > **Pasta src:** possui o código fonte do projeto, arquivos .css e .html
